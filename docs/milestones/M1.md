# M1 - Android Optical Flow Module Skeleton

## Goal
Build Android-side optical flow pipeline skeleton:

Camera frame -> OpenCV LK -> flow_x/flow_y/quality -> CSV log -> MAVLink sender stub

## Out of scope
- PX4 EKF fusion
- Real flight validation
- ToF/range fusion on Android

## Definition of Done
- Camera preview runs
- Analyzer receives frames continuously
- LK outputs flow_x, flow_y, quality
- Static scene outputs near-zero flow
- Hand translation produces consistent direction
- CSV log format is fixed and documented
- MAVLink sender stub interface callable
- CI remains green

## Deliverables
- Android skeleton app (CameraX + analyzer)
- LK estimator module
- quality metric definition
- CSV logger
- sender stub interface
- testdata spec and sample logs
